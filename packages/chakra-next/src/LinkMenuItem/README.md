# LinkMenuItem

Combines a Chakra `MenuItem` and a Next `Link`

Accepts an optional `nextLinkProps` passed down to the Next `Link`.

```jsx
import { LinkMenuItem } from '@v1v2/chakra-next'

const App = () => (
  // ...
  <LinkMenuItem href="/about" color="red">
    About
  </LinkMenuItem>
  // ...
)
```

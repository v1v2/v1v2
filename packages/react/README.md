# @v1v2/react

React components and hooks

## Installation

```sh
npm i @v1v2/react
# or
yarn add @v1v2/react
```

## Components

- [**Defer**](src/Defer#readme) – Defers the rendering of a component (`useDefer` is also provided)
- [**ConditionalWrapper**](src/ConditionalWrapper#readme) – Wraps content conditionally
